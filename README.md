## StockDataCollector - 股票資料收集器

這個程式是一個股票資料收集器，用於收集個股日成交資訊。

### 資料來源
你可以從以下網址獲取資料：[https://www.twse.com.tw/zh/trading/historical/stock-day.html](https://www.twse.com.tw/zh/trading/historical/stock-day.html)

### 操作方式
1. 下載 `.exe` 檔案
2. 輸入要的股票代號以及需要的月份量
3. wait(防止被鎖抓資料有間隔2秒)
4. done
